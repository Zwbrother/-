- **技术栈**: 采用 Vite + React 18 + React Router v6 架构，UI 组件库为 Ant Design 5，状态管理使用 Zustand。
- **路由与权限**: `App.jsx` 定义全局路由，通过 `Protected` 组件实现基于角色（STUDENT/COUNSELOR/ADMIN）的路由守卫，未登录或角色不匹配时重定向至登录页。
- **布局体系**: 采用组合式布局设计，`BaseLayout` 封装通用的侧边栏、顶部导航及用户信息展示逻辑；`StudentLayout`、`AdminLayout` 等具体布局通过注入菜单配置复用 `BaseLayout`。
- **数据交互**: `api.js` 封装 Axios 实例，统一处理 baseURL、超时设置及请求/响应拦截器（自动附加 Token、处理 401 过期跳转及全局错误提示）。
- **状态管理**: `store.js` 使用 Zustand 配合 persist 中间件实现认证状态（Token、User）的持久化存储。
