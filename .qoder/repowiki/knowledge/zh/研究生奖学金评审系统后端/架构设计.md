- **分层架构**：采用经典的 Controller-Service-Mapper 结构，通过 MyBatis-Plus 进行数据持久化。
- **安全机制**：自定义 `JwtAuthInterceptor` 拦截器配合 `@RequireRole` 注解实现基于角色的访问控制（RBAC），利用 ThreadLocal (`AuthContext`) 管理用户会话。
- **业务逻辑**：核心评分引擎封装在 `ScoreCalcService` 中，支持品德、专业素质及五大能力模块的复杂加权计算；`ScholarshipService` 处理资格判定与奖金发放规则。
- **统一响应**：通过 `GlobalExceptionHandler` 和 `R<T>` 泛型类实现标准化的 API 响应格式与异常处理。
